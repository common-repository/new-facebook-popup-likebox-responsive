It Will Show a popup like box when a user visits your WordPress site!


== Installation ==
Goto to Plugin page and Upload the zip . Activate plugin & goto settings to SetUp your Own page ULR.

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
